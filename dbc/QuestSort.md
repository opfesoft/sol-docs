# QuestSort

**QuestSort.dbc**

This DBC contains all quest sorts that can be used as negative values in the field [quest_template.QuestSortID](../db/world/quest_template.md#questsortid).

**Version is : 3.3.5a**

## Structure

| Column | Field | Type |
| --- | --- | --- |
| 1 | ID | Integer |
| 2-18 | sRefName | String + [Loc](Localization_lang.md) |

## Description of the fields

### QuestID

The Quest Id from [quest\_template.id](quest_template.md#id)

### id

Used to group multiple entries from quest\_poi\_points.id it is the id of the POI.

### ObjectiveIndex

if -1 than position of npc where you can complete quest

### MapID

The Map id from [Map.dbc](../../dbc/Map.md)

### WorldMapAreaId

The ID from WorldMapArea.dbc.

### Floor

This is the AreaID of the POI.

### Priority

`field-no-description|7`

### Flags

`field-no-description|8`

## Description of the fields

### guid

Player's GUID. See [characters.guid](characters.md#guid)

### setguid

First free guid.

### setindex

Set index, values from 0 to 9 are used.

### name

Individual. Name is set by player.

### iconname

Name taken from ItemDisplayInfo.dbc, column 6

### ignore\_mask

`field-no-description|6`

### item0-18

Values taken from item\_instance.guid

     0. Head

1.  Neck
2.  Shoulder
3.  Shirt
4.  Chest
5.  Waist
6.  Legs
7.  Feet
8.  Wrist
9.  Hands
10. Ring 1
11. Ring 2
12. Trinket 1
13. Trinket 2
14. Back
15. Main Hand
16. Off Hand
17. Relic
18. Tabard

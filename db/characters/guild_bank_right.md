## Description of the fields

### guildid

The ID of the guild. See [guild.guildid](guild.md#guildid)

### TabId

The ID of the Tab you are setting the permissions for. See [guild\_bank\_tab.TabId](guild_bank_tab.md#tabid)

### rid

The rank you are setting the permissions for. See [guild\_rank.rid](guild_rank.md#rid)

### gbright

The permissions you want to give to a player of that rank on the tab. This is a bitmask. To combine permissions, you must do the OR operation sum the flags.

Example:

``` cpp
UPDATE `guild_bank` SET `gbright`=`gbright`|4 WHERE...)
```

FLAGS:

1 = view items

2 = deposit items

4 = update item name shown when navigating the tab

8 = withdraw items

255 = Has all rights

### SlotPerDay

The number of items that a player can withdraw per day (if permissions give him the right to withdraw items).
